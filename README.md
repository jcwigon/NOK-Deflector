# NOK Detector

## Overview
Automated inspection system for detecting defective products (NOK) using deep learning. The model analyzes images and classifies them as OK or NOK.

## Features
- Image classification using PyTorch
- 600+ training images
- Web interface with Streamlit
- Easy to use and deploy

## Installation
```bash
git clone https://github.com/jcwigon/nok-detector.git
cd nok-detector
